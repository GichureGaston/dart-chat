# Realtime Chat App in Dart

Dart Chat is a CLI chat client that communicates over TCP , a connection oriented protocol .

## Getting Started

first terminal - start the server

    dart run bin/main.dart

second and third terminal - connect clients

    dart run bin/chat_client.dart

## Checking the TCP Connection

to verify the TCP connection is open run

    lsof -i :5000

you should see something like this

    COMMAND   PID    USER   FD   TYPE DEVICE SIZE/OFF NODE NAME
    dart     1234   user   10u  IPv4  12345      0t0  TCP *:5000 (LISTEN)

if you see the above output the server is running and listening on port 5000

## Login

in the 2nd and 3rd terminals login with the following command

    login <userId> <userName> <roomId>

User 1 - 2nd terminal

    login user1 alice room101

User 2 - 3rd terminal

    login user2 bob room101

both users must join the same roomId to communicate . on success you will see

    [LOGIN] Welcome to the chat, alice!
    [PRESENCE] bob is online

## Sending Messages

    message <userId> <roomId> <text>

User 1 sends a message

    message user1 room101 Hey bob, are you there?

User 2 replies

    message user2 room101 Hey alice, I am here!

the other user will see

    [user1] Hey bob, are you there?

messageId is automatically generated by the client so you dont need to worry about it

## Quit

    quit

disconnects from the server

## Commands

login <userId> <userName> <roomId> - login and join a chatroom
message <userId> <roomId> <text> - send a message to a room
quit - disconnect from the server

## Notes

- server runs on localhost:5000
- both users must be in the same roomId to exchange messages
- presence updates are broadcast automatically when a user joins or leaves
